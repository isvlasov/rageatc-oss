# CLAUDE.md Discipline

CLAUDE.md is a **lean pointer file** - orientation for a fresh session, not a knowledge base. It contains only: project identity (one line), pointers to key files, session resumption steps, and project-specific working conventions.

If information lives (or should live) in another file, CLAUDE.md **points to it - never absorbs it**: architecture -> ARCHITECTURE.md, requirements -> PRD.md, current status -> STATUS.md, build instructions -> README.md, history -> git.

CLAUDE.md changes when project structure changes, not when project state changes.
